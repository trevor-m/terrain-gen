# terrain-gen
Procedurally generates a terrain using Perlin noise and displays it with OpenGL.

### Features
* Texture mixing
* Lighting (Ambient, Diffuse and Specular)


### Libraries Used
* OpenGL 3.3
* GLFW 3
* GLEW
* SOIL
* GLM
